# Tag-game
Play Game
https://yuki-katayama.github.io/Tag-game/index.html

## 日本語
### 概要
制限時間の間、敵から逃げ続けるゲームです。

### ルール
+ 20秒おきに、敵が一匹ずつ増える。

アイテム
+ 塩 → 幽霊を撃退
+ 玉ねぎ → ドラキュラを撃退
+ 豆 → 鬼を撃退

キー操作
+ [A, W, S, D] or [←, →, ↑, ↓]

### コードの変更可能点
game.jsで、以下の点が変更可能
+ アイテムの出現サイクルの頻度
+ 制限時間
+ 鬼が追加されるタイミング
+ 敵の追跡スピード etc..
